# Another MONday! — Weekly Whiteboard

A minimal digital weekly whiteboard for **Bemnet** and **Nati**. Built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Two person panels** — Click your icon to expand and see your task list (Bemnet: sage green accent, Nati: dusty blue accent).
- **Tasks** — Add tasks, mark them done with a checkbox, and edit or delete them.
- **Another Monday** — Clears all task lists for a fresh week.
- **Previous Mondays** — Placeholder for future history (e.g. Firebase).

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** — Off-white background, sage green & dusty blue palette
- **Framer Motion** — Floating icons, expand/collapse, list animations
- **Lucide React** — Icons (User, UserCircle, Plus, Pencil, etc.)
